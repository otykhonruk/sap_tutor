# Базові поняття системи ЛІС (SAP) щодо руху та зберігання майна

## Номери партій

+-----------------------------------+--------------------------------------------------------------------------------------------+
| **Номер партії**                  | **Пояснення**                                                                              |
+===================================+============================================================================================+
| **1000000000**                    | Майно I категорії, без визначеного року вислуги, що знаходиться на зберіганні на складах.  |
+-----------------------------------+--------------------------------------------------------------------------------------------+
| **2000000000**                    | Майно II категорії, без визначеного року вислуги, що знаходиться на зберіганні на складах. |
+-----------------------------------+--------------------------------------------------------------------------------------------+
| **200000XXXX**,                   | Майно II категорії, із визначеним роком вислуги (крайні 4 цифри).                          |
|                                   |                                                                                            |
| де XXXX означає рік вислуги майна | Майно з цією партією може бути як видане у використання, так і знаходитись на складах.     |
|                                   |                                                                                            |
| Наприклад: 2000002025             |                                                                                            |
+-----------------------------------+--------------------------------------------------------------------------------------------+
| **3000000000**                    | Майно III категорії, що знаходиться на зберіганні на складах.                              |
+-----------------------------------+--------------------------------------------------------------------------------------------+

## Коди віртуальних складів

+----------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **2090** | Майно у зберіганні на складах в/частини.                                                                                                                                                           |
|          |                                                                                                                                                                                                    |
|          | Сюди найчастіше відноситься майно з партіями 1000000000 та 2000000000, але також може відноситись й вживане майно із закінченим терміном використання (200000ХХХХ, де ХХХХ – рік вислуги майна). |
+==========+====================================================================================================================================================================================================+
| **209U** | Майно II категорії, видане у використання.                                                                                                                                                         |
|          |                                                                                                                                                                                                    |
|          | Сюди відноситься майно з партіями 200000ХХХХ (де ХХХХ означає рік вислуги майна).                                                                                                                  |
+----------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **209W** | Майно у недоторканому запасі.                                                                                                                                                                      |
+----------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
