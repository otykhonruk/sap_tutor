# Інші звіти у системі SAP

## Список документів матеріалів (проведених операцій), 

Системний звіт \"Список документів матеріалів\" відображає список операцій, які були проведені з майном у системі SAP. Завдяки багатим можливостям фільтрування та початкових критеріїв, з цим звітом ви зможете знайти будь-яку операцію, яку ви проводили в системі, та дізнатися її номер та інші деталі.

Отримана у звіті інформація знадобиться у багатьох випадках. Наприклад, коли вам потрібно дізнатись додаткові деталі проведеної операції, або сторнувати її (що неможливо, не знаючи номер операції).

Для того, щоб сформувати звіт зі списком документів матеріалів (проведених операцій) у системі SAP, виконайте наступні кроки.

1\. Увійдіть у систему та відкрийте вікно \"Робоче місце користувача\".

Див. розділ [\"Початкове вікно роботи з системою\"](#початкове-вікно-роботи-з-системою).

2\. Натисніть кнопку-кокпіт \"Звіт: Список документів матеріалів \[CP0004\]\".

![A screenshot of a computer Description automatically generated](media/media/image178.png){width="6.299212598425197in" height="2.4645669291338583in"}

3\. У початковому вікні операції \"Список документів матеріалів\", у полі \"Завод\", вкажіть номер вашого заводу (4-значний код вашої в/частини у системі).

![A screenshot of a computer Description automatically generated](media/media/image179.png){width="4.282288932633421in" height="3.0978258967629047in"}

Додатково, вкажіть будь-які інші критерії, щоб звузити пошук серед проведених операцій.

Наприклад, вкажіть код матеріалу (найменування майна). Ви можете скопіювати код матеріалу з еЗвіту або зі створеного плану потреб.

Якщо серед критеріїв пошуку вказано матеріал, то система відобразить всі операції, проведені виключно з цим матеріалом.

![A screenshot of a computer Description automatically generated](media/media/image180.png){width="4.739130577427821in" height="2.8434787839020124in"}

4\. Вказавши всі критерії пошуку, натисніть кнопку ![](media/media/image32.png){width="0.2222222222222222in" height="0.20833333333333334in"} \"Виконати\".

![A screenshot of a computer Description automatically generated](media/media/image181.png){width="3.5304352580927385in" height="2.103238188976378in"}

5\. У вікні \"Список документів матеріалів\" буде відображено всі операції, які відповідають критеріям пошуку, заданим у кроці 3.

![A screenshot of a computer Description automatically generated](media/media/image182.png){width="6.299212598425197in" height="1.031496062992126in"}

Задача виконана: звіт зі списком проведених операцій сформовано.

\*\*\*

**Щоб скопіювати вміст будь-якої чарунки** у сформованому звіті проведених операцій, виконайте наступні кроки:

1\. У сформованому звіті \"Список документів матеріалів\", знайдіть рядок з потрібною вам операцією, та натисніть його у будь-якому місці, щоб виділити.

![A screenshot of a computer Description automatically generated](media/media/image182.png){width="6.299212598425197in" height="1.031496062992126in"}

2\. Натисніть одночасно клавіші \"Ctrl\" та \"Y\" на клавіатурі комп\'ютера та поставте курсор у потрібну чарунку.

3\. Натисніть одночасно клавіші \"Ctrl\" та \"C\" на клавіатурі комп\'ютера.

![A screenshot of a computer Description automatically generated](media/media/image183.png){width="6.299212598425197in" height="0.9291338582677166in"}

Задача виконана: вміст чарунки скопійовано у буфер обміну комп\'ютера.
