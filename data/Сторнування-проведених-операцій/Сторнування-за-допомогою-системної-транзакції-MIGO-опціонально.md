## Сторнування за допомогою системної транзакції MIGO (опціонально)

**!!** Для сторнування рекомендується використовувати операцію-кокпіт \"Сторнування документів матеріалів \[CP0443\]\", оскільки вона є простішою у використанні; така простота знижує вірогідність помилок, які можуть зробити користувачі при виконанні.

Системна транзакція MIGO є набагато більш комплексною та складнішою для використання. Сторнування за допомогою MIGO описане в якості виключення, як додатковий спосіб сторнування.

### 1. Запустіть транзакцію MIGO

1.1. Перейдіть до початкового вікна роботи з системою, \"SAP Easy Access\".

Див. розділ [\"Початкове вікно роботи з системою\"](#початкове-вікно-роботи-з-системою).

1.2. Запустіть транзакцію \"migo\". Відкриється вікно \"Скасування документу матеріалу\".

Див. розділ [\"Виконання транзакцій за їх системним кодом\"](#введення-номеру-заводу-з-кодом-маскування).

![A screenshot of a computer Description automatically generated](media/media/image198.png){width="4.304348206474191in" height="2.4033431758530184in"}

### 2. Проведіть сторнування обраних позицій найменування

2.1. У вікні \"Скасування документу матеріалу\", у меню в верхньому лівому куті вікна, виберіть тип операції: \"Скасування\".

![A screenshot of a computer Description automatically generated](media/media/image199.png){width="5.278261154855643in" height="2.303698600174978in"}

2.2. У полі для номеру операції (третє від меню з типом операції), вставте з буферу обміну номер операції. Щоб вставити номер, поставте курсор у поле та натисніть одночасно клавіші \"Ctrl\" та \"V\" на клавіатурі комп\'ютера.

Номер операції можна дізнатись за допомогою операції \"Звіт: Список документів матеріалів\". Див. розділ [\"Підготовчі кроки перед сторнуванням\"](#підготовчі-кроки-перед-сторнуванням).

![A screenshot of a computer Description automatically generated](media/media/image200.png){width="4.817391732283465in" height="1.981913823272091in"}

2.3. Натисніть клавішу \"Enter\" на клавіатурі комп\'ютера. У середній частині вікна \"Скасування документу матеріалу\" з\'являться рядки з матеріалами (найменуваннями майна), проведеними у обраній операції.

![A screenshot of a computer Description automatically generated](media/media/image201.png){width="6.299212598425197in" height="3.7401574803149606in"}

2.4. В нижній третині вікна \"Скасування документу матеріалу\", натисніть кнопку ![](media/media/image202.png){width="0.19639326334208224in" height="0.21509733158355204in"} \"Закрити дані подр.\". Нижню частину вікна з детальними даними буде приховано.

Цей крок потрібен, щоб легко виділити потрібні матеріали (найменування майна) для сторнування у наступному кроці.

![A screenshot of a computer Description automatically generated](media/media/image203.png){width="4.230849737532808in" height="1.8347823709536308in"}

2.5. У таблиці з матеріалом (або матеріалами) всередині вікна, натисніть пустий квадрат у колонці \"ОК\", щоб обрати матеріал. У квадраті з\'явиться відмітка-прапорець; він означатиме, що рядок з майном обраний для сторнування.

![A screenshot of a computer Description automatically generated](media/media/image204.png){width="4.230555555555555in" height="1.6753215223097113in"}

2.6. Натисніть кнопку \"Провести\" у верхній частині вікна. Обраний рядок або рядки з майном повинні зникнути з таблиці всередині вікна.

![A screenshot of a computer Description automatically generated](media/media/image205.png){width="4.230555555555555in" height="2.039394138232721in"}

Задача виконана: у системі відбулося сторнування (скасування) руху обраного найменування майна, який був проведений у відповідній операції.
