## Підготовчі кроки перед сторнуванням

**1. Дізнайтесь код матеріалу (найменування майна), операцію з яким потрібно сторнувати, та скопіюйте його.**

Цей крок не є обов\'язковим; він потрібен, щоб швидше знайти операцію, яку потрібно сторнувати. Ви можете одразу перейти до кроку 2 (пошук номеру операції для сторнування), але тоді пошук потрібної операції у списку всіх операцій, які ви провели у системі, може зайняти набагато більше часу.

1.1. Увійдіть у систему та відкрийте вікно \"Робоче місце користувача\".

1.2. Сформуйте еЗвіт.

Див. розділ [\"Формування еЗвіту у системі ЛІС (кроки)\"](#формування-езвіту-у-системі-ліс-кроки).

1.3. Знайдіть рядок з матеріалом (найменуванням майна), операцію з яким потрібно сторнувати.

1.4. Поставте курсор у чарунку з кодом матеріалу та скопіюйте код, одночасно натиснувши клавіші \"Ctrl\" та \"C\" на клавіатурі комп\'ютера.

![A screenshot of a computer Description automatically generated](media/media/image184.png){width="3.9391305774278216in" height="1.502843394575678in"}

Крок 1 виконано – ви скопіювали код матеріалу у буфер обміну комп\'ютера.

**2. Дізнайтесь номер операції з руху майна, яку потрібно сторнувати, та скопіюйте його.**

Для пошуку операції з руху майна, використайте операцію \"Звіт: Список документів матеріалів\". Для того, щоб зменшити перелік відображених операцій, в якості критеріїв пошуку (фільтрів) вкажіть номер заводу (4-значний код вашої в/частини у системі) та код матеріалу, операцію з яким потрібно сторнувати (ви скопіювали його у кроці 1).

Зауважте, в системі можуть бути автоматичні операції по вашому заводу, які ви не проводили, наприклад надходження майна від ОЦЗ. Їх можна ідентифікувати по користувачу (SYS_BATCH або S0\*). Не сторнуйте такі операції! У разі питань по автоматичних проводках звертайтесь до підтримки.

2.1. Увійдіть у систему та відкрийте вікно \"Робоче місце користувача\".

Див. розділ [\"Початкове вікно роботи з системою\"](#початкове-вікно-роботи-з-системою).

2.2. Натисніть кнопку-кокпіт \"Звіт: Список документів матеріалів \[CP0004\]\".

![A screenshot of a computer Description automatically generated](media/media/image178.png){width="6.299212598425197in" height="2.4645669291338583in"}

2.3. У початковому вікні операції \"Список документів матеріалів\", вкажіть такі дані:

\- у полі \"Матеріал\", вставте з буферу обміну код матеріалу (майна). Для цього, поставте курсор у поле, потім одночасно натисніть клавіші \"Ctrl\" та \"V\" на клавіатурі комп\'ютера.

\- у полі \"Завод\", вкажіть номер вашого заводу (4-значний код вашої в/частини у системі)

![A screenshot of a computer Description automatically generated](media/media/image180.png){width="4.739130577427821in" height="2.8434787839020124in"}

2.4. Вказавши всі критерії пошуку, натисніть кнопку ![](media/media/image32.png){width="0.2222222222222222in" height="0.20833333333333334in"} \"Виконати\".

![A screenshot of a computer Description automatically generated](media/media/image181.png){width="3.5304352580927385in" height="2.103238188976378in"}

2.5. У вікні \"Список документів матеріалів\", знайдіть рядок з операцією, яку потрібно сторнувати, та натисніть його у будь-якому місці, щоб виділити.

![A screenshot of a computer Description automatically generated](media/media/image182.png){width="6.299212598425197in" height="1.031496062992126in"}

2.6. Натисніть одночасно клавіші \"Ctrl\" та \"Y\" на клавіатурі комп\'ютера та поставте курсор у потрібну чарунку у стовпці \"Документ матеріалу\".

2.7. Натисніть одночасно клавіші \"Ctrl\" та \"C\" на клавіатурі комп\'ютера.

![A screenshot of a computer Description automatically generated](media/media/image183.png){width="6.299212598425197in" height="0.9291338582677166in"}

Крок 2 виконано – ви скопіювали номер операції, яку потрібно сторнувати, до буферу обміну комп\'ютера.
