## Формування еЗвіту у системі ЛІС (кроки)

1. Увійдіть у систему ЛІС.

> Див. деталі у розділі [\"Вхід до системи\"](#вхід-до-системи-загальні-кроки).

2. Відкрийте вікно \"Робоче місце користувача\".

Див. розділ [\"Початкове вікно роботи з системою\"](#початкове-вікно-роботи-з-системою).

3. Натисніть кнопку-кокпіт \"Наявність та рух речового майна \[CP0135\]\".

![A screenshot of a computer Description automatically generated](media/media/image69.png){width="6.379630358705162in" height="1.3490857392825897in"}

4. У вікні \"Критерії вибору\", вкажіть такі дані:

| **Завод**  | 4-значний код вашої в/частини у системі.|Наприклад: 4000. 
-------------------------------------------------------
|                    | Цей номер, дійсний для вашого облікового запису, завжди зазначений на верхній панелі вікні \"Робоче місце користувача\".                                                                                  |
|                    |                                                                                                                                                                                                           |
|                    | ![A screenshot of a computer AI-generated content may be incorrect.](media/media/image50.png){width="3.0277777777777777in" height="1.2366262029746282in"}                                                 |
|                    |                                                                                                                                                                                                           |
|                    | Після введення коду, система автоматично замінить введений номер на код маскування. Це очікувана поведінка системи; див. розділ [\"Введення номеру заводу\"](#введення-номеру-заводу-з-кодом-маскування). |
+====================+===========================================================================================================================================================================================================+
| **Звітний період** | Дати, які повинен охоплювати звіт.                                                                                                                                                                        |
|                    |                                                                                                                                                                                                           |
|                    | Наприклад: якщо формуєте річний звіт, вкажіть дати з 01.01.20ХХ по 31.12.20ХХ, де \"20ХХ\" вказує звітний рік.                                                                                            |
|                    |                                                                                                                                                                                                           |
|                    | Див. також розділ [\"Річні плани потреб\"](#річні-плани-потреб).                                                                                                                                          |
+--------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

5. Натисніть кнопку ![](media/media/image70.png){width="0.25in" height="0.25in"} \"Виконати\" (або натисніть клавішу F8 на клавіатурі комп\'ютера).

![A screenshot of a computer Description automatically generated](media/media/image71.png){width="4.462963692038495in" height="1.4231988188976379in"}

Сформований еЗвіт відображатиметься у вікні \"Звіт щодо наявності та руху речового майна за звітний період\".

![A screenshot of a computer Description automatically generated](media/media/image72.png){width="6.299212598425197in" height="2.97244094488189in"}
