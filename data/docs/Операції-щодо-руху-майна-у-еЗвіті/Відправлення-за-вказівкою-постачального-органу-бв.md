## Відправлення за вказівкою постачального органу (б/в)

### Типи майна

У операції "Відправити за вказівкою ПО (б/в)" обліковується:

\- речове майно ІІ категорії, яке в звітному періоді відправлено за вказівкою постачального органу іншим ОЦЗ або в/частинам (які не являються постачальним органом для звітуючої в/частини);

\- речове майно ІІ категорії, яке здано в звітному періоді на ОЦЗ (постачальний орган).

### Кроки проведення операції

**1. Сформуйте еЗвіт.**

> Див. детальні кроки у розділі [["Формування еЗвіту у системі ЛІС"]{.underline}](#формування-езвіту-у-системі-ліс-кроки).

**2. Запустіть операцію.**

2.1. У вікні еЗвіту, виділіть рядок (або декілька рядків) з майном, з яким потрібно провести операцію.

Щоб виділити рядок, натисніть лівою кнопкою миші на сірий квадрат з лівого боку потрібного рядку. Обраний рядок змінить колір на жовтий.

![A screenshot of a computer Description automatically generated](media/media/image97.png){width="6.425336832895888in" height="1.0260870516185476in"}

Щоб виділити декілька рядків, розташованих поруч, протягніть натиснутий курсор мишки вниз чи вверх, щоб захопити потрібні рядки.

Щоб виділити декілька рядків, не розташованих поруч, після виділення одного рядку, натисніть клавішу "Ctrl" (Control) та, утримуючи її натиснутою, виділіть інші рядки, один за одним.

![A screenshot of a computer Description automatically generated](media/media/image98.png){width="6.425in" height="1.2201301399825022in"}

2.2. Натисніть стрілку-трикутник ◢ у правому боці кнопки ![](media/media/image99.png){width="1.0833333333333333in" height="0.2222222222222222in"} та у контекстному меню оберіть "Відправити за вказівкою ПО (б/в)".

![A screenshot of a computer Description automatically generated](media/media/image143.png){width="6.299212598425197in" height="2.7716535433070866in"}

Або, у рядку з потрібним матеріалом у еЗвіті, у колонці "ІД" натисніть піктограму ![A pixelated image of a fruit Description automatically generated](media/media/image100.png){width="0.19641951006124234in" height="0.20869531933508312in"} та оберіть "Відправити за вказівкою ПО (б/в)".

Якщо потрібно провести операцію руху одразу з декількома матеріалами:

\- Оберіть рядки з потрібними матеріалами у еЗвіті.

\- Натисніть стрілку-трикутник на правому боці кнопки ![](media/media/image99.png){width="0.8746380139982503in" height="0.17941272965879265in"} та оберіть "Відправити за вказівкою ПО (б/в)".

**3. Вкажіть дані проводки та проведіть операцію.**

3.1. У полі "Дата проводки", вверху вікна операції, вкажіть дату впродовж поточного або попереднього місяця.

Див. розділ ["Дата проводки операції"](#дата-проводки-для-операцій-з-руху-майна) для детальних рекомендацій.

3.2. Вкажіть дані для кожного найменування майна у відповідних полях проводки операції:

+----------------+------------------------------------------------------------------------------------------------------------------------------------------+
| **ДатаПерДок** | Вкажіть дату **накладної** як первинного облікового документу, згідно якого операція з матеріалом була здійснена фактично.               |
|                |                                                                                                                                          |
|                | Наприклад: 01.03.2024                                                                                                                    |
+================+==========================================================================================================================================+
| **№ПервДок**   | Вкажіть **назву та номер** **накладної** як первинного облікового документу, згідно якого операція з матеріалом була здійснена фактично. |
|                |                                                                                                                                          |
|                | Наприклад: Накладна 889/123                                                                                                              |
+----------------+------------------------------------------------------------------------------------------------------------------------------------------+
| **Контрагент** | Вкажіть **номер в/частини або назву установи**, в яку було передано майно.                                                               |
|                |                                                                                                                                          |
|                | Наприклад: А0001                                                                                                                         |

| **Примітка**   | Вкажіть номер та дату розпорядження, виданого органом постачання або ЦУРЗ, на відправлення майна.                                        |
|                |                                                                                                                                          |
|                | Наприклад: Розпорядження на відправлення майна №89/12 від 01.03.2024                                                                     |

**4. Проведіть операцію у системі.**

4.1. Після введення даних, натисніть піктограму ![](media/media/image110.png){width="0.15625in" height="0.1736111111111111in"} в правому нижньому куті вікна операції.

Якщо операція була проведена у системі успішно, у нижньому лівому куті з'явиться зелена відмітка та повідомлення про номер операції у системі ЛІС.

### Первинні облікові документи

1\. Накладна, ТА

2\. Розпорядження на передачу майна (вкажіть номер та дату такого розпорядження у полі "Примітка" при проведенні операції).

### Результати проведення операції у системі

1\. Проведене майно (з партії 2000000000) не переводиться на жодний з віртуальних складів в/частини у системі SAP (ані на 2090, ані на 209U). Натомість, майно повністю списується з балансу в/частини у системі, а потреба у майні зростає.

2\. Збільшиться кількість майна у таких графах (колонках) еЗвіту: 19, 30.

3\. Зменшиться кількість майна у графах (колонках) еЗвіту: 24, 29.
