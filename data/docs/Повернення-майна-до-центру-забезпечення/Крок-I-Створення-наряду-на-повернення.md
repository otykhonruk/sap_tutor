## Крок I: Створення наряду на повернення

**1. Увійдіть до системи ЛІС (SAP).**

Див. деталі у розділі [["Вхід до системи"]{.underline}](#вхід-до-системи-загальні-кроки).

**2. Запустіть операцію для створення наряду на повернення або обмін.**

2.1. Відкрийте вікно "Робоче місце користувача".

2.2. Натисніть кнопку-кокпіт "Наряд на повернення/обмін CP\[0420\]".

![A screenshot of a computer Description automatically generated](media/media/image241.png){width="6.268055555555556in" height="2.6770833333333335in"}

Зауважте, що розташування кнопки-кокпіту може відрізнятись для різних користувачів.

**3. Вкажіть критерії для проведення операції (найменування майна та ОЦЗ-отримувач).**

3.1. У вікні "Наряд на повернення/обмін", на вкладці "Критерії відбору", заповніть необхідні поля, вказуючи наступну інформацію:

**- Матеріал (**або **NSN код матеріалу):** код найменування майна, яке потрібно обробити у операції.

Коди майна можна дізнатись з облікового документу, згідно якого ваша в/частина отримала відповідне майно від ОЦЗ.

Якщо найменувань майна для повернення декілька, скористайтесь функцією множинного вибору для відповідного поля ("Матеріал" або "NSN код матеріалу").

При виборі матеріалу завжди обирайте максимальну деталізацію, із розміром, якщо майно розмірне, типом, видом, класом. Використовуйте пошук, щоб підібрати найбільш точний відповідник. Не повертайте розмірне майно без розмірів!

Наприклад, ви повертаєте панаму літню польову 54 розміру, і у пошуку бачите матеріали «ПАНАМА ЛП», «ПАНАМА ЛП Р54», та кілька матеріалів на кшталт «ПАНАМА ЛП Т1 К7 Р54», з типом і класом. З цього списку бажано підібрати найбільш точний (довгий) варіант, з відповідним розміром, типом і класом.

**- Завод-постачальник:** код ОЦЗ у системі ЛІС (SAP), на який ви повертатимете майно.

![A screenshot of a computer Description automatically generated](media/media/image242.png){width="4.730435258092738in" height="2.953246937882765in"}

3.2. Натисніть вкладку "Обрані записи", щоб продовжити роботу із обраним майном.

**4. Додайте майно до вікна (кошику) обробки.**

У вікні "Наряд на повернення/обмін", на вкладці "Обрані записи", у верхній частині вікна, кожне найменування обраного майна (матеріалу) відображається окремим рядком.

4.1. Оберіть (виділіть) рядок з потрібним найменуванням майна. Для цього, натисніть лівою кнопкою миші на сірий квадрат з лівого боку потрібного рядку. Обраний рядок змінить колір на жовтий.

Щоб виділити декілька рядків, розташованих поруч, протягніть натиснутий курсор мишки вниз чи вверх, щоб захопити потрібні рядки.

Щоб виділити декілька рядків, не розташованих поруч, після виділення одного рядку, натисніть клавішу "Ctrl" (Control) та, утримуючи її натиснутою, виділіть інші рядки, один за одним.

![A screenshot of a chat Description automatically generated](media/media/image243.png){width="4.1043482064741905in" height="1.2377613735783026in"}

4.2. Натисніть кнопку ![](media/media/image193.png){width="0.1985695538057743in" height="0.21842629046369205in"} у правій частині вікна з обраними записами. Або, двічі натисніть рядок з потрібним майном лівою кнопкою миші.

Якщо ви повертаєте на ОЦЗ декілька найменувань майна, виділіть всі рядки з майном та перемістіть їх до вікна обробки, натиснувши кнопку ![](media/media/image193.png){width="0.1985695538057743in" height="0.21842629046369205in"}.

![A screenshot of a computer Description automatically generated](media/media/image244.png){width="6.268055555555556in" height="1.9555555555555555in"}

Обране майно буде відображено у нижній частині вікна "Наряд на повернення/обмін"; ця частина вікна називається "Вікном (кошиком) обробки".

![A screenshot of a computer Description automatically generated](media/media/image245.png){width="6.268055555555556in" height="2.890277777777778in"}

**5. Для кожного найменування майна у вікні обробки, введіть потрібні дані.**

Дані потрібно вводити у полях білого кольору. Поля іншого кольору недоступні для редагування.

Щоб ввести дані, натисніть у відповідному полі та введіть потрібне значення. Для введення, використовуйте клавіатуру комп'ютера або вставляйте скопійоване значення з буферу обміну, одночасно натискаючи клавіші "Ctrl" та "V".

| **Назва поля**      | **Що вказати**                                                                                                      |
+=====================+=====================================================================================================================+
| Ціна повернення     | Вкажіть ціну за одиницю, з якою майно було оприбутковано у вашій в/частині.                                         |
|                     |                                                                                                                     |
|                     | Ціну можна взяти з облікового документу, згідно якого ваша в/частина отримала відповідне майно від ОЦЗ.             |

+---------------------+---------------------------------------------------------------------------------------------------------------------+

**6. Створіть наряд на повернення.**

6.1. У панелі інструментів над вікном (кошиком) обробки, натисніть кнопку "Повернення". Всі найменування майна у вікні обробки будуть додані до проводки.

5.3. Натисніть кнопку ![](media/media/image249.png){width="0.20347222222222222in" height="0.20347222222222222in"} "Виконати".

Якщо операція проведена успішно, у системі буде створено документ "Наряд на повернення".

Наряд на повернення відноситься до типу системних документів "Документ закупівлі".

**Номер створеного наряду** буде відображено у стовпці "Документ закупівлі" (нижня частина вікна "Наряд на повернення/обмін").

**!! Обов'язково занотуйте цей номер для подальшої роботи у системі.** Як мінімум, номер знадобиться для Кроку II – друку Акту приймання-передачі майна.

![A screenshot of a computer Description automatically generated](media/media/image250.png){width="6.268055555555556in" height="0.7284722222222222in"}

**Відкрити детальну інформацію про наряд на повернення**: у вікні "Наряд на повернення/обмін" натисніть номер наряду у стовпці "Документ закупівлі". Потрібна інформація буде відображена у детальному вікні "Наряд на повернення...".

Переглянути деталі наряду також можна за допомогою транзакції *me23n*: по замовчуванню, транзакція відображає ОСТАННІЙ наряд, який ви створили в системі.

![A screenshot of a computer Description automatically generated](media/media/image251.png){width="6.268055555555556in" height="2.317361111111111in"}

**Скопіювати код наряду:** за допомогою курсору, виділіть номер у полі справа від меню "Наряд на повернення", та одночасно натисніть кнопки "Ctrl" та "C" на клавіатурі комп'ютера.

**Дізнатись статус погодження** (розблокування) наряду з боку ОЦЗ (офіцеру ВТЗРМ): у детальному вікні "Наряд на повернення..." натисніть вкладку "Стратегія розблокування". Жовтий трикутник означає, що наряд очікує погодження; зелений – що погодження надано.
