## Перегляд створеного плану потреб

Щоб переглянути створений план потреб, виконайте наступні кроки.

1\. Увійдіть у систему SAP (LIS).

Див. розділ ["Вхід до системи"](#вхід-до-системи-загальні-кроки).

2\. Відкрийте вікно "Робоче місце користувача.

Див. розділ ["Початкове вікно роботи з системою"](#початкове-вікно-роботи-з-системою).

**3. Запустіть операцію "Перегляд потреб (спрощений)".**

3.1. У вікні "Робоче місце користувача", натисніть кнопку-кокпіт "Спрощений облік: потреби \[CP0116\].

![A screenshot of a computer Description automatically generated](media/media/image31.png){width="6.299212598425197in" height="1.3385826771653544in"}

3.2. Оберіть рядок "Перегляд потреб (спрощений)", натиснувши його один раз лівою кнопкою миші.

3.3. Натисніть кнопку ![](media/media/image32.png){width="0.2222222222222222in" height="0.20833333333333334in"} "Виконати" (або ліворуч від рядку, або у панелі задач під назвою вікна "Відобразити область меню YPLSIMPL01").

![A screenshot of a computer Description automatically generated](media/media/image58.png){width="2.990741469816273in" height="1.629915791776028in"}

Запустити перегляд плану потреб також можна з вікна еЗвіту. Див. розділ ["Перегляд плану потреб з вікна еЗвіту"](#формати-відображення-даних-у-езвіті).

4\. На сторінці "Перегляд плану" ви побачите створений план потреб з усією необхідною інформацією.

![A screenshot of a computer Description automatically generated](media/media/image59.png){width="6.299212598425197in" height="1.7007874015748032in"}

Якщо на сторінці "Перегляд плану" план не відобразився автоматично, у полі "План" вкажіть номер плану та натисніть "Enter" на клавіатурі комп'ютера.\
\
Якщо ви не пам'ятаєте номер плану, оберіть план зі списку можливих:

4.1. Натисніть кнопку ![](media/media/image36.png){width="0.25in" height="0.2638888888888889in"} праворуч від поля "План"

4.2. Знайдіть запис потрібного плану у списку.

У колонці "Завод" поруч з потрібним планом має бути зазначений номер вашого заводу (4-значний код вашої в/частини у системі).

![A screenshot of a computer Description automatically generated](media/media/image60.png){width="3.3941141732283464in" height="2.6574070428696412in"}

4.3. Двічі натисніть потрібний запис.

4.4. На сторінці "Перегляд плану", натисніть "Enter" на клавіатурі комп'ютера, щоб відобразити план.

![A screenshot of a computer Description automatically generated](media/media/image59.png){width="6.299212598425197in" height="1.7007874015748032in"}
