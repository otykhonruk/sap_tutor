## Річні плани потреб

**Огляд.** В системі ЛІС, для кожного звітного року необхідно створити та підтримувати ОКРЕМИЙ план потреб.

Окремі плани потрібні для того, щоб чітко звітувати за результатами кожного року, та щоб показники одного року не мішались із показниками іншого.

**Кроки роботи з планами потреб** детально описані вище у таких розділах цього Посібника:

\- [Ведення планових показників (вибір норм та чисельності особового складу)](#ведення-планових-показників-вибір-норм-та-чисельності-особового-складу)

\- [Створення плану потреб](#створення-плану-потреб)

**Часові параметри планових показників.** Система ЛІС створює річні плани потреб на основі наступних параметрів у планових показниках:

1\. Дійсно з (Дата з)

2\. Дійсно по (Дата до)

НАПРИКЛАД:

ЯКЩО у планових показниках параметр \"Дійсно з\" = 01.01.2024 та параметр \"Дійсно по\" = 31.12.2024,

ТО система створить за такими показниками план потреб на 2024 рік,

АЛЕ не зможе створити з цими показниками план на 2025 рік.

**Ведення однакових норм для різних річних планів.**

При введенні планових показників для декількох планів, уважно слідкуйте за наступним:

\- Для кожного річного плану потреб, створюйте окремі рядки з плановими показниками

\- Якщо ви використовуєте однакові норми забезпечення для двох та більше річних планів (що трапляється дуже часто), то рядки з плановими показниками повинні вказувати одну й ту саму норму, але РІЗНІ дати дійсності (\"Дійсно з\" та \"Дійсно по\") та різну кількість о/складу.

НАПРИКЛАД:

ЯКЩО ви використовуєте Норму №1 для планів 2024 та 2025 років,

ТО у операції \"Ведення планових показників\" додайте ДВА рядки з плановими показниками:

ДЛЯ плану потреб **2024** року, рядок з Нормою №1 повинен мати наступні показники:

\- \"Норма\" = \"РЕЧ_01\" (або його відповідники),

\- \"Дійсно з\" = \"01.01.2024\"

\- \"Дійсно до\" = \"31.12.2024\"

\- \"Значення\" = кількість о/складу для 2024

ДЛЯ плану потреб **2025** року, рядок з Нормою №1 повинен мати наступні показники:

\- \"Норма\" = \"РЕЧ_01\" (або його відповідники), як і для 2024;

\- \"Дійсно з\" = \"01.01.2025\"

\- \"Дійсно до\" = \"31.12.2025\"

\- \"Значення\" = кількість о/складу для 2025

**Обґрунтування змін.** При створенні першої версії річного плану потреб, у полі \"Обґрунтування останніх змін\", можна зазначити:

\"Створення нового плану потреб за ХХХХ рік\",\
де ХХХХ – конкретний звітний рік (наприклад, 2025).

**Рекомендації щодо зручної роботи з плановими показниками для річних планів.**

У операції \"Ведення планових показників\", щоб відобразити показники ТІЛЬКИ за певний рік, скористайтесь функцією \"Встановити фільтр\".

НАПРИКЛАД:

Для відображення показників за 2024 рік:

\- Для стовпця **\"Дійсно з\"**, встановіть фільтр на **\"01.01.2024\"**.\
- Для стовпця **\"Дійсно до\"**, встановіть фільтр на **\"31.12.2024\"**.

Таким саме чином, можна встановити фільтри на інші поля, для вашої зручності.

**Річні плани у еЗвіті.** Для того, щоб еЗвіт працював на основі певного річного плану, достатньо сформувати еЗвіт САМЕ за цей рік.

НАПРИКЛАД:

ЯКЩО ви хочете працювати у еЗвіті з планом потреб 2025 року,\
ТО при формуванні еЗвіту вкажіть дати в межах 2025 року,\
НАПРИКЛАД, з 01.01.2025 по 31.12.2025.
